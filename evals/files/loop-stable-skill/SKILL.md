# Loop Stable Skill

## Defaults

- Start with `SKILL.md`, `README.md`, and `evals/evals.json`.
- Touch support files only when they would otherwise become stale.
- Stop when another pass would only be cosmetic.

## QA

- Keep a local `.gitignore`.
- Keep evals aligned with behavior.
- Link helper files directly from the skill when they matter.
